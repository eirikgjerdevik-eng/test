How to play

Make words using the 7 letters. Every word must include the center letter (highlighted in gold). Letters can be reused.

You start with 60 seconds. Each valid word adds time:

4 letters → +5s
5 letters → +8s
6 letters → +10s
7+ letters → +15s

Find as many words as you can before time runs out!
